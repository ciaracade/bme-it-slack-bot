# Univeristy of Michigan Biomedical Engineering IT Department Slack Bot

A chat bot that seamlessly integrates [TeamDynamix](https://teamdynamix.com), [OpenAI](https://openai.com/), and [Slack](https://slack.com) to provide efficient ticket assistance for the University of Michigan Biomedical Engineering IT Department.

## Todo

## Features
- Users can create, view, update, and close tickets directly via Slack.
- Automatic ticket status updates and notifications within Slack.
- Utilize OpenAI to provide instant, context-aware responses to user queries.
- Suggest relevant articles or solutions to assist IT staff.
- Secure and seamless API integration with TeamDynamix and Slack.
- Auto-classify and route tickets based on content.
- Send automated follow-ups and reminders for unresolved tickets.
